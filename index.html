Welcome to Pointing-utils documentation!

Project description

This library provides utilities to deal with pointing data

Installing Pointing utils

The library can be installed directly with pip:

python3 -m pip pointing-utils

Alternatively, you can also downlaod the source and build it using poetry

poetry install

Using pointing_utils

only one class available for now

  • FittsModel –> used to compute Fitts’ law regressions and associated throughputs
